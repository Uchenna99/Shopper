SHOPPER

=> Overview

The Shopper App is a full-stack e-commerce application that allows users to browse products, manage carts, and place orders seamlessly.
It is built to reflect real-world shopping workflows, focusing on performance, usability, and clean architecture.

=> Key Features
> User authentication
> Product listing and search
> Shopping cart functionality
> Order creation and management
> Responsive UI for mobile and desktop
> Secure API endpoints

=> Tech Stack

> Frontend - React, TypeScript, Tailwind CSS
> Backend - Node.js, Express
> Database - PostgreSQL managed with Prisma

=> Other Tools
> REST APIs
> JWT authentication

=> Application Flow

> User signs up or logs in
> Browses available products
> Adds items to cart
> Places an order
> Tracks order status



=> Getting Started

> Prerequisites - Node.js, PostgreSQL database

> Installation
```js
git clone https://github.com/Uchenna99/Shopper.git
cd Shopper
npm install
```

> Run Locally
```js
npm run dev
```
